# Ludo Game

A simple multiplayer Ludo game simulation with turn-based movement and dice roll logic.

## What this project does
- Simulates a multiplayer Ludo board
- Generates random dice roll values
- Applies valid movement rules for each token

## Tech Stack
- Python (pygame) OR JavaScript (choose one if you used it)
- Basic OOP logic

## Features
- Turn-based gameplay
- Dice roll simulation
- Token movement rules
